# House Price Prediction 🏠

Machine Learning project to predict house prices using house features.

## Project Overview

This project uses Machine Learning to predict the price of a house based on:

- Area
- Bedrooms
- Bathrooms
- Stories
- Parking

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Machine Learning Model

Algorithm used:

- Linear Regression
